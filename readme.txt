Convert WooCommerce product gallery thumbnails in a responsive carousel with mouseover zoom effect.

== Description ==

Add a better way to showcase your single products with the carousel and zoom effect. Overing product image, zoom effect will be activated and the image gallery will be displayed like a carousel improving the user experience.

[youtube https://www.youtube.com/watch?v=JQYx4OJujpY]

= Plugin Features =
* Select the zoom position above or right of the featured image.
* Customize Carousel pagination and navigation colors.
* Change the default product image lightbox effect with the zoom one.
* Select the number of thumbnails to show in the carousel.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for `woo-product-carousel-and-zoom`
3. Click 'Install Now'
4. Activate the plugin on the Plugins dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `woo-product-carousel-and-zoom.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugins dashboard

= Using FTP =

1. Download `woo-product-carousel-and-zoom.zip`
2. Extract the `woo-product-carousel-and-zoom` directory in your computer
3. Upload the `woo-product-carousel-and-zoom` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugins dashboard
